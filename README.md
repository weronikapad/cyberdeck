# Raspberry pi pico cyberek

## Function:
It's a retro inspired minicomputer, and hopefully a debugger, built on a raspberry pi pico, rn the files only present a mother board but i will be adding a keybord, screen, sd card, and i LiPo battery

## Sepecifics:
-Raspbery pi pico (Raspberry_Pi_Pico)
-USB-C port (USB_C_Receptable_USB.2.0_14P)
-Power manager (TP4057)
-Battery connector (Conn_01x02)
-screen connector (J2 Conn_01x08)
-column connector (J5 Conn_01x08)
-keyboard connector (Conn_01x04)

## Files
-/gerber/ PCB munfactoring files
-/cyberek_schematic.kicad.sch KiCad schematic
-/cyberdeck_pcb.kicad_pcb KiCad PCB layout file
-/cyberdeck_pcb.kicad_pro General PCB KiCad file
